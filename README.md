# 3D-modeling-assistant

ブラウザで直接開けるシングルページの 3D モデラーです。`index.html` をダブルクリックするだけで起動し、直方体・球・円柱の配置や、グリッド上で描いたポリゴンの押し出しに対応しています。オブジェクトは移動・回転・スケールをハンドルで編集でき、シーンは JSON または glTF で保存 / 読み込みできます。

## 使い方

1. リポジトリをダウンロードし、`index.html` をブラウザで開きます（ローカルサーバー不要）。
2. 左パネルの「作成」で基本形状を追加するか、「平面ポリゴン描画」でグリッド上に頂点を置き「面を押し出す」を実行します。
3. 選択したオブジェクトは移動 / 回転 / スケールハンドルで編集できます（削除キーで削除）。
4. 編集補助: ビュープリセット（アイソメ / 上 / 前 / 右）、スナップ移動、寸法表示、複製、色変更に対応しています。
5. 「シーンをJSON保存」または「gltfを書き出し」で保存できます。読み込みは「JSONを開く」から行います。glTF (.gltf / .glb) の読み込みにも対応しています。

> three.js とその拡張（TransformControls / GLTFExporter）は CDN から取得します。オフラインで使う場合は `app.js` 内の import パスをローカルに保存したファイルへ変更してください。

### SKP (SketchUp) について

SKP は SketchUp 固有のバイナリ形式のため、ブラウザ単体でのネイティブ読込・書出しには対応していません。必要な場合は、glTF または JSON を外部ツールで SKP に変換してください。
